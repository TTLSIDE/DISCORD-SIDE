How to use
Log In with token Paste a Discord token and press Log In. The token is validated before login. Requires discord.com in the active tab.
Extract Token Pull your session token from an active Discord tab. Works with any login method.
Saved Accounts Click the bookmark icon in the top right to save the current token. Click a saved account to load it, or use the ▶ icon for instant login.
Speed Mode Toggle in ⚙ Settings. Disables all animations — panels, overlays, and transitions become instant. Great for fast account switching.
Shortcuts Ctrl+Shift+L — Log In   Ctrl+Shift+E — Extract   Ctrl+Shift+C — Copy token
